## ⏰ 每日时间安排

> **核心原则**：午休公园跑 · 不占早晨送娃时间 · 不影响晚上哄睡

| 时间段 | 周一（开会）/ 周四（休息） | 周二 / 周三 / 周五（公园跑） |
|--------|--------------------------|------------------------------|
| 🌅 早上 | 起床、送孩子上学，正常节奏不受影响 | 同左，照常送娃上学 |
| 💻 9:30–12:00 | 专注工作 | 专注工作 |
| 🏃 12:00–13:10 | 午饭 + 休息（不跑步，充分放松） | 换跑鞋出发（3min）→ 公园跑道跑步（25~30min）→ 回来换衣擦汗（5min）→ 吃午饭（剩余时间） |
| 😴 13:10–14:00 | 午休小憩 20分钟（有助于下午专注力） | 午休小憩 15~20 分钟 |
| 💻 14:00–19:30 | 下午专注工作 | 下午工作（运动后精神状态更好！） |
| 🌙 19:30–21:30 | 回家陪娃 · 吃饭 · 哄睡，家庭时间完全不占用 | 同左 |
| 🧘 21:30 之后 | 个人时间、拉伸 10 分钟、早睡 | 跑步日：重点做腿部拉伸 5~8 分钟，然后早些休息保证恢复 |

## 💡 实用技巧

> 降低执行门槛，把跑步变成自然习惯。

| 类别 | 具体建议 |
|------|---------|
| 👟 装备放公司 | 跑鞋 + 换洗衣服 + 毛巾放在公司固定位置，12 点直接出门，零准备时间 |
| 🏃 公园跑道好处 | 距离固定、环境好、安全，比街道更适合建立配速感和距离感 |
| ❤️ 配速判断 | **「说话测试」**：跑步时能说完整句子 = 强度合适；上气不接下气 = 太快了，慢下来 |
| ⏱️ 午休时间管理 | 建议 12:00 换装出门，12:05 开跑，12:35 回到公司，给自己充足的吃饭和休息时间 |
| 🦵 跑后拉伸 | 每次跑完必做：小腿拉伸 + 大腿前侧拉伸，各 30秒 × 2，能显著减少久坐后的酸痛 |
| 💧 补水 | 跑前 15 分钟喝 200ml 水；跑后尽快补水；公园跑不需要带水，时间短 |
| 📱 推荐 App | **Keep** 或 **悦跑圈**：自动记录每次距离/配速/心率，几周后看数据进步很有成就感 |
| 🌧️ 下雨天 | 跳过跑步没关系；或在公司走廊快走 20 分钟 + 做深蹲/俯卧撑代替 |
| ⚡ 减量周 | 每 4 周第 4 周跑量降约 30%，这不是偷懒，是让身体适应、避免受伤的必要安排 |
| 😴 睡眠优先 | 如果前一晚严重睡眠不足，当天改成快走或直接休息。跑步是为了更好的状态 |
| 📈 4 周后的变化 | 爬楼梯不再喘 · 下午不再犯困 · 睡眠质量提升——比体重下降更早出现的进步信号 |
| 🎯 周一心态 | 周一开会不跑很正常！这是计划的一部分，不是缺席。每周 4 次已经很高效了 |
